# DevoraClip-Releases
 # DevoraClip v2.0.4

A modern clipboard manager for Windows designed to make clipboard history faster, smarter, and more productive.

## ✨ What's New in v2.0.4

* Enhanced text viewing experience
* Improved search functionality
* Better clipboard content rendering
* UI refinements and usability enhancements
* General bug fixes and optimizations
* Improved RTF and HTML content handling
* Better navigation and clipboard content management

## 🚀 Features

### Clipboard Management

* **Clipboard History** – Automatically captures and stores clipboard entries for quick access anytime.
* **Quick Copy & Reuse** – Restore previous clipboard entries with a single click.
* **Local Data Storage** – Clipboard history remains on your device.
* **Full Content Viewer** – Open large clipboard items in a dedicated window for easier reading and editing.

### Content Format Support

* **Plain Text, RTF & HTML Support** – View clipboard content in multiple formats and switch between them instantly.
* **Rich Text (RTF) Viewer** – Preserve and inspect rich text formatting without losing content fidelity.
* **HTML Viewer** – Render and preview HTML clipboard content.
* **Plain Text Editor** – Edit, format, and clean up plain text clipboard content on the fly.

### Search & Productivity

* **Advanced Search** – Search within clipboard content and quickly navigate between matches.
* **Keyboard-Friendly Navigation** – Quickly browse and manage clipboard entries using keyboard shortcuts.
* **Fast Content Discovery** – Instantly find previously copied information.

### Text Transformation Tools

* **UPPERCASE Conversion**
* **lowercase Conversion**
* **Title Case Conversion**
* **Sentence Case Conversion**
* **Trim Whitespace**
* **Additional Text Cleanup Utilities**

### Modern Experience

* **Modern Windows UI** – Clean, native Windows design built for Windows 10 and Windows 11.
* **Self-Contained Deployment** – No separate .NET runtime installation required.
* **Portable Mode** – Run without installation; all data stays local alongside the executable.
* **Optimized Performance** – Fast startup and responsive clipboard history browsing.

## 💼 Portable Mode

1. Extract the ZIP file.
2. Ensure `portable.flag` exists beside `DevoraClip.exe`.
3. Launch `DevoraClip.exe`.

All application data, settings, and history will be stored locally within the portable folder.

## ✔ Requirements

* Windows 10 Version 19041 or later
* Windows 11
* 64-bit Windows (x64)

## 📝 Notes

> Existing users are encouraged to back up their data before upgrading.

* This is a self-contained release and does not require a separate .NET installation.
* Portable mode keeps all application data local to the application folder.
* No administrator privileges are required for portable usage.
